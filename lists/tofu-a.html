1. Online scrap metal recycling

Create an online commerce platform for consumers with excess scrap metal components that they would like to get rid of. This allows customers to convert metal for no longer need for cash. Revenues are from margin on transactions.

32. DIY web application creator

Create a web that lets you create other simple web apps, or prototypes of them at least. Instead of having to program in code the notion of different database tables and attributes for conceptual models in a web application system, this web app would let you create “models” in a simple web UI, give them “attributes” and establish basic relationships between different types of models—as well as picking certain “views” as items in a menu bar, and allowing businesses to prototype applications that might be useful to them very quickly and without having to touch any code. This web app could even export a simple prototype web application code (e.g. a Ruby on Rails application skeleton) that can be handed off to a design/development team to polish up into a fully fledged app.

33. Physical address proxy service

When you move house, doing the mail redirection is always a pain. The postal service site you have to use is usually terrible, you have to renew it after a few months if you haven’t managed to track down all the companies that still have your old address on file, and it’s hard to keep track of which organizations are still sending you stuff (even after you ask them not to!). This business is a proxy service, where you pay this company for a virtual physical address that you can hand out to banks, insurance companies, etc.—basically any organization that you suspect might make it hard to change your address in the future—and then you tell this company what your current address is via an incredibly well-designed user interface, meaning that you only have one place you need to change it next time you move. This service could provide extra value by allowing you to track the “from” companies that are sending you mail, and press “pause” via the web interface to not have that mail forwarded to you any longer—especially useful for companies that don’t stop mailing you!

34. Concierge setup of third-party digital tools

Create a business that is positioned as an expert in all the third-party tools available, their capabilities, their integration points, and sell that knowledge to businesses as a concierge service tailored to each specific client’s business goals.
